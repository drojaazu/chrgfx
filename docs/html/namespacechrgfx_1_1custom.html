Detailed Description

This namespace should be used for any non-standard tile, palette or color converters.

By non-standard, we mean converters that can not use chrdefs/paldefs/coldefs.

For example, tiles that do not have a regular bit pattern or psuedo-retro file formats such as palettes for graphics editors.
